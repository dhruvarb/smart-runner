# Contributing

Thanks for helping improve Smart Runner.

## Development Setup

```bash
npm install
npm run compile
npm test
```

Use VS Code's Extension Development Host to run the extension locally.

## Pull Request Checklist

- Keep changes focused on one behavior or sprint goal.
- Add or update tests for shared logic.
- Run `npm run compile`.
- Run `npm test`.
- Update `README.md` or `CHANGELOG.md` when behavior changes.
