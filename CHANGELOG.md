# Changelog

## 0.0.1

- Created the initial VS Code extension foundation.
- Added `Smart Runner: Run Selected Snippets`.
- Added TypeScript build configuration and project documentation.
- Added the Sprint 2 selection engine for collecting and merging selected snippets.
- Added the Sprint 3 temporary file manager with unique file creation and cleanup.
- Added the Sprint 4 language registry for Python, JavaScript, and TypeScript.
- Added the Sprint 5 execution engine with stdout, stderr, and exit-code capture.
- Added Sprint 6 terminal integration with reusable VS Code terminal execution.
- Added Sprint 7 editor title, context menu, and keyboard shortcut contributions.
- Added Sprint 8 command support for C, C++, Java, Go, and Rust.
- Added Sprint 9 VS Code configuration settings and wired core settings into execution.
- Added Sprint 10 same-file dependency resolution for obvious function and class references.
- Added Sprint 11 current-function execution when no code is selected.
- Added Sprint 12 persisted execution history, rerun-last, and history picker commands.
- Added Sprint 13 unit test infrastructure and focused tests for execution commands and temporary files.
- Added Sprint 14 marketplace packaging script, VSIX ignore rules, and release documentation.
- Added Sprint 15 BYO-key AI integration with provider settings, Secret Storage key management, AI workflow commands, and local provider support.
- Improved dependency resolution to include variable/import dependencies automatically and warn when referenced names are not found.
- Fixed Windows temp path quoting for Git Bash terminals.
- Added Smart Runner to the editor run dropdown and prioritized its editor title play button.
- Added AI CodeLens actions for selected code, an Explain Selection status-bar button, and AI quick fixes for diagnostics.
- Added Smart Runner AI help panel and startup activation so AI selection helpers register immediately.
- Added visible Run Selection CodeLens and status-bar buttons for selected code.
- Added visible Explain Error CodeLens and status-bar buttons for selected code.
- Added setup documentation and project banner.
